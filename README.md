## Running the project

``` bash
npm install
npm start
# To run the project in watch mode use the following command
npm run watch
```

If you have [SAM Local](https://github.com/awslabs/aws-sam-cli) installed you can run the following command to run this as a local lambda

``` bash
npm run serverless
```
